# Thesis (LaTeX, Overleaf-ready)

LaTeX source for the dissertation, structured as one file per chapter and using IEEE-style
numbered referencing (`\bibliographystyle{ieeetr}`).

## Uploading to Overleaf

1. Zip the contents of this folder so that `main.tex` sits at the root of the zip (not inside an
   extra subfolder).
2. On Overleaf: **New Project ‚Üí Upload Project**, and select the zip file.
3. Set the compiler to **pdfLaTeX** (Menu ‚Üí Compiler). Every package used here ships with a
   standard TeX Live install, so no further configuration is needed.
4. If Overleaf asks which file is the root document, choose `main.tex`.

## Table of contents, list of figures, list of tables

Generated automatically by `\tableofcontents`, `\listoffigures`, and `\listoftables` in
`main.tex`. They update on every recompile ‚Äî after adding a new chapter, section, figure, or
table, compile twice so the page numbers settle. This is standard LaTeX behaviour, not a bug.

## Figures and labelling

Figures use the standard `figure` environment with a `\caption{}` and `\label{}`, which produces
the automatic "Figure 3.1: ..." numbering and populates the List of Figures. Reference a figure
in text with `\cref{fig:...}` rather than typing the number by hand, so it stays correct if
chapters are reordered.

The six figures included are generated by the companion notebook at
`../research/notebooks/research.ipynb`:

1. `01_label_distribution.png` ‚Äî rewrite-operation label counts by data split
2. `02_length_and_compression.png` ‚Äî sentence length and compression ratio distributions
3. `03_complex_vs_simple_scatter.png` ‚Äî complex vs. simplified sentence length
4. `04_training_loss_curve.png` ‚Äî fine-tuning training/validation loss
5. `05_metric_comparison.png` ‚Äî readability/SARI/BLEU/ROUGE/BERTScore comparison across systems
6. `06_sari_vs_bertscore.png` ‚Äî SARI vs. BERTScore trade-off scatter

If the notebook is re-run (for example, after the full-scale training run described in
`chapters/08_conclusion.tex`), copy the regenerated PNGs from `../research/outputs/figures/`
into this folder's `figures/`, keeping the same filenames, and the figures update automatically
on the next compile. The tables in Chapters 3, 4, and 7 are typed directly into the `.tex` files
rather than generated automatically, so if the underlying numbers change, update those tables by
hand from `../research/outputs/tables/*.csv`.

## IEEE referencing

Citations use the standard `\cite{key}` command, with the bibliography style set in `main.tex`:

```latex
\bibliographystyle{ieeetr}
\bibliography{references}
```

`ieeetr` ships with every standard LaTeX distribution, including Overleaf, so no extra package or
`.bst` file is required. It produces numbered in-text citations (`[1]`, `[2]`, ...) in order of
first appearance, with a matching numbered reference list at the end of the document.

To add a reference: add a BibTeX entry to `references.bib`, then cite it with `\cite{yourkey}` in
any chapter file.
